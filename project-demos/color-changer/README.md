# Color Changer

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Description

Color Changer is a simple interactive app demonstrating dynamic styling and state management in Ivy. Click any color swatch to update the preview panel in real time.

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```

## Tags

Starter, Demo, Color, State Management
