# Vc 

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy). 

Ivy is a web framework for building interactive web applications using C# and .NET.

## Description

CRM-VC is a Customer Relationship Management system for venture capital firms with dashboard analytics, startup management, and investment deal tracking.

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```
