# RentalBackOffice 

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy). 

Ivy is a web framework for building interactive web applications using C# and .NET.

## Description

Rental Back Office is a property rental management system for managing properties, tenants, leases, payments, and maintenance requests.

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```
