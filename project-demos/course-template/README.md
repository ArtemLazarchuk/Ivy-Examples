# Course Template

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy). 

Ivy is a web framework for building interactive web applications using C# and .NET.

## Description

Course Template is a template for creating interactive online courses by automatically converting Markdown files into interactive course pages with navigation and structured content.

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```
